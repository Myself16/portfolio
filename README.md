# 🌐 Portfolio

A minimal, fast-loading portfolio built with **vanilla HTML, CSS, and JavaScript** to showcase projects and technical work without unnecessary dependencies.

---

## 🧠 Philosophy

This project avoids heavy frameworks to focus on:

* performance
* simplicity
* full control over structure and behavior

The goal is not just to display projects, but to demonstrate clean fundamentals.

---

## ⚙️ Tech Stack

* **HTML5** — semantic structure
* **CSS3** — layout, responsiveness, styling
* **JavaScript** — interactivity and dynamic behavior

No frameworks. No build tools. Just core web fundamentals.
